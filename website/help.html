Help

Bluetooth Troubleshooting

- Make sure you are using the chrome browser (it supports web bluetooth)
- Make sure bluetooth is turned on on your device

Saving data

- Clicking the save data button will save the current recorded data locally to your computer
- .csv files can be used in excel, google sheets, nz grapher, and many other external tools
for further analysis

SHM experiments

- If your SHM data doesn't ressemble a nice, clean sine wave, have a think about the neccesary
conditions for SHM. Why might this result in messy data?
