Bluetooth Troubleshooting

  • Make sure you are using the Chrome browser (it supports Web Bluetooth).
  • Ensure Bluetooth is turned on on your device.

Saving Data

  • Clicking the save data button will save the current recorded data locally to your computer.
  • .csv files can be used in Excel, Google Sheets, NZ Grapher, and many other external tools for further analysis.

SHM Experiments

  • If your SHM data doesn't resemble a clean sine wave, consider the necessary conditions for simple harmonic motion (SHM). Why might this result in messy data?
